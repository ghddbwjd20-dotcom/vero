# 섹션별 동영상 배경 및 블러 전환 효과 가이드

## 🎬 구현된 기능

### 1. **동영상 배경 시스템**
- **VideoBackground 컴포넌트**: 재사용 가능한 동영상 배경 컴포넌트
- **자동 폴백**: 동영상 파일이 없을 때 CSS 애니메이션으로 대체
- **성능 최적화**: HEAD 요청으로 동영상 존재 여부 확인
- **에러 처리**: 동영상 로드 실패 시 자동 폴백

### 2. **섹션별 동영상 배경**
- **프로세스 섹션**: `/videos/process-background.mp4`
- **고객 후기 섹션**: `/videos/testimonials-background.mp4`
- **Hero 섹션**: `/videos/hero-background.mp4` (기존)

### 3. **블러 전환 효과**
- **섹션 간 부드러운 연결**: 각 섹션의 경계에서 블러 효과
- **동영상 섹션 전용**: 동영상 배경 섹션에 특별한 블러 효과
- **다크 섹션 대응**: 어두운 배경 섹션용 블러 효과

## 🛠️ 기술적 구현

### 1. **VideoBackground 컴포넌트**

```tsx
interface VideoBackgroundProps {
  videoSrc: string                    // 동영상 파일 경로
  fallbackGradient?: string          // 폴백 그라데이션
  overlay?: boolean                  // 오버레이 표시 여부
  blur?: boolean                     // 동영상 블러 효과
  className?: string                 // 추가 CSS 클래스
  children?: React.ReactNode         // 자식 컴포넌트
  blurTransition?: boolean           // 블러 전환 효과
}
```

### 2. **동영상 배경 로직**

```tsx
// 동영상 파일 존재 여부 확인
useEffect(() => {
  const checkVideo = async () => {
    try {
      const response = await fetch(videoSrc, { method: 'HEAD' })
      setHasVideo(response.ok)
    } catch (error) {
      setHasVideo(false)
    }
  }
  checkVideo()
}, [videoSrc])
```

### 3. **블러 전환 CSS**

```css
/* 동영상 배경 섹션용 블러 전환 */
.video-section-blur::before {
  content: '';
  position: absolute;
  top: -30px;
  left: 0;
  right: 0;
  height: 60px;
  background: linear-gradient(
    180deg,
    rgba(199, 164, 106, 0) 0%,
    rgba(199, 164, 106, 0.1) 50%,
    rgba(199, 164, 106, 0.2) 100%
  );
  backdrop-filter: blur(15px);
  z-index: 1;
  pointer-events: none;
}
```

## 🎨 시각적 효과

### 1. **프로세스 섹션**
- **동영상**: 프로세스 관련 동영상 배경
- **폴백**: 베로 골드/프라이머리 색상의 그라데이션
- **블러**: 섹션 경계에서 부드러운 블러 전환
- **오버레이**: 동영상 위에 반투명 오버레이

### 2. **고객 후기 섹션**
- **동영상**: 고객 후기 관련 동영상 배경
- **폴백**: 프라이머리/골드 색상의 그라데이션
- **블러**: 동영상 자체에 블러 효과 적용
- **전환**: 섹션 간 부드러운 블러 전환

### 3. **섹션 간 전환**
- **상단 블러**: 이전 섹션과의 부드러운 연결
- **하단 블러**: 다음 섹션과의 부드러운 연결
- **색상 조화**: 각 섹션의 색상에 맞는 블러 효과

- **에러 처리**: 로드 실패 시 즉시 폴백

### 3. **메모리 관리**
- **자동 정리**: 컴포넌트 언마운트 시 정리
- **이벤트 리스너**: 메모리 누수 방지
- **상태 관리**: 불필요한 리렌더링 방지

## 🔧 커스터마이징

### 1. **동영상 교체**
```tsx
<VideoBackground
  videoSrc="/videos/your-video.mp4"
  fallbackGradient="bg-gradient-to-br from-blue-500 to-purple-500"
  overlay={true}
  blur={false}
  blurTransition={true}
>
  {/* 콘텐츠 */}
</VideoBackground>
```

### 2. **블러 효과 조정**
```css
/* 블러 강도 조정 */
.video-section-blur::before {
  backdrop-filter: blur(20px); /* 더 강한 블러 */
}

/* 블러 높이 조정 */
.video-section-blur::before {
  height: 80px; /* 더 높은 블러 영역 */
}
```

/* 다른 색상 테마 */
